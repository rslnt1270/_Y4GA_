Importar historial
Opcion A — Subir JSON
📂
Toca para seleccionar el archivo
uber_viajes_YYYY_MM.json
Opcion B
Extractor automatico (Python):
1. Instala: pip install playwright y playwright install chromium
2. Descarga y ejecuta el script.
3. El navegador se abre. Navega por tu historial de ganancias.
4. Presiona ENTER cuando termines — el JSON se guarda automaticamente.
5. Sube el archivo con la Opcion A de arriba.
Opcion A — Subir JSON
📂
Toca para seleccionar el archivo
didi_viajes_YYYY_MM.json
Opcion B
Extractor automatico para DiDi (Python):
1. Instala: pip install playwright y playwright install chromium
2. Descarga y ejecuta el script.
3. El navegador abre el portal de conductores DiDi.
4. Navega por Ganancias > Historial semana a semana.
5. Presiona ENTER al terminar — el JSON se guarda automaticamente.
6. Sube el archivo con la Opcion A de arriba.

Nota: DiDi cambia su estructura frecuentemente. Si no se capturan datos, verifica que navegaste por la seccion de historial/ganancias.
